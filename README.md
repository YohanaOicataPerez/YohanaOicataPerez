### 🎯 RFM Customer Segmentation — BigQuery + Power BI
> End-to-end customer segmentation pipeline built on 12 months of sales data using Google BigQuery, visualized in Power BI for marketing decision-making.
- Consolidated 12 monthly sales tables into a unified dataset using `UNION ALL`
- Calculated Recency, Frequency, and Monetary metrics per customer with window functions
- Assigned decile scores (1–10) and combined them into an RFM total score
- Segmented customers into 8 business-ready tiers: Champions, Loyal VIPs, Potential Loyalist, Promising, Engaged, Requires Attention, Risk, and Lost/Inactive
- Built a BI-ready view connected to a Power BI dashboard for marketing team use
